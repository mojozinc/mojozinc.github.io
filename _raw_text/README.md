# Raw Writings & Source Drafts

This directory contains the raw text, notes, and drafts used to write essays for this portfolio site.

## Why the folder name starts with an underscore (`_`)

By default, GitHub Pages uses Jekyll to build and serve the static files in this repository. Jekyll has a built-in rule: **any file or folder starting with an underscore is ignored and will not be processed or served.**

This means that while these drafts are safely committed and backed up to GitHub (where they are visible in your repository's files), they are **not** accessible via a web URL on your live website (e.g., `mojozinc.github.io/_raw_text/...` will return a 404 error).

### Getting Started

Place your raw markdown or text drafts here, and convert them to the HTML template in the `posts/` folder when you are ready to publish!
